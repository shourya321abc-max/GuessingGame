Number Guessing Game (Java)

A simple and interactive console-based number guessing game built in Java. The program generates a random number between 1 and 100, and the player must guess it with guidance provided after each attempt.

 Features
 
     Random number generation (1–100)
     Continuous guessing loop until correct answer
     Tracks number of attempts
     Input validation (handles non-numeric input safely)

Feedback system:
    
     "Too High"
     "Too Low"
     "Correct Guess"  

Technologies Used

     Java 
     java.util.Scanner for user input
     java.util.Random for number generation

How to Play

       The system randomly selects a number between 1 and 100.
       Enter your guess in the console.
       The program will guide you:
              Too low → Guess higher
              Too high → Guess lower
      Keep guessing until you find the correct number.
      The program displays the total number of attempts.
